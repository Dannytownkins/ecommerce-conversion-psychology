# Ethics Gate — Check FIRST, Override ALL

These rules are non-negotiable. They override every other recommendation in this plugin. Based on EU DSA Art 25, FTC Fake Reviews Rule (Oct 2024), CA SB-478 (Honest Pricing), CA SB-1001 (Bot Disclosure), and Epic Games $245M penalty precedent.

## Urgency / Scarcity

- Countdown timers MUST be tied to real, verifiable deadlines — else BLOCK
- Scarcity claims ("Only X left") MUST reflect actual current inventory — else BLOCK
- "People viewing" counters MUST reflect real concurrent sessions — else BLOCK
- Fake urgency backfires: 17.6% sales DECREASE observed from fabricated "only X left" messages

## Pricing

- First displayed price MUST include all mandatory non-government fees — else BLOCK (CA SB-478)
- Strikethrough "was" prices MUST be real previous selling prices — else BLOCK
- Shipping costs MUST be disclosed before payment info collection — else BLOCK
- 39-48% of cart abandonment is caused by hidden costs revealed at checkout

## Reviews / Social Proof

- All reviews MUST be from real users of the product — else BLOCK (FTC Fake Reviews Rule)
- Insider/affiliate reviews MUST be clearly labeled — else BLOCK
- Negative reviews MUST NOT be suppressed or de-ranked — else BLOCK
- Penalty: up to $51,744 per violation (FTC)

## Choice Architecture

- Accept/decline options MUST be visually symmetrical — else BLOCK (EU DSA Art 25)
- No confirm-shaming language ("No thanks, I don't want to save money") — BLOCK
- No pre-checked boxes adding items or services — BLOCK
- Penalty: up to 6% global annual turnover (EU DSA)

## Override Rule

When in doubt, recommend the ethical alternative. Dark patterns are not just unethical — they are increasingly illegal and measurably counterproductive.
