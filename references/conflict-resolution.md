# Conflict Resolution — When Principles Conflict

Priority order (highest wins):

1. **Legal/accessibility compliance** — WCAG AA, EU DSA, FTC, CA SB-478. Non-negotiable.
2. **Ethics gate** — No dark patterns, even if they "convert better."
3. **User-specified constraints** — The user has business context you don't. Respect stated constraints.
4. **Domain-specific guidance** — Research-backed recommendations from loaded references.

## Intra-Level Tiebreaker

The principle from the higher-priority reference file (as ordered in the routing table) wins.

## Example

If pricing psychology says "use charm pricing ($9.99)" but the user says "we're a luxury brand" — user constraint wins. Recommend round pricing ($10).
